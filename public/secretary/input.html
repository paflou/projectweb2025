Εισαγωγή Δεδομένων από JSON

Οδηγίες

Μπορείτε να εισάγετε δεδομένα φοιτητών και καθηγητών από αρχείο JSON. Το αρχείο πρέπει να έχει την ακόλουθη δομή:

{
  "students": [
    {
      "name": "Όνομα",
      "surname": "Επώνυμο",
      "student_number": "1043XXXX",
      "email": "email@students.upatras.gr",
      "username": "username",
      "password": "password",
      "street": "Οδός",
      "number": "Αριθμός",
      "city": "Πόλη",
      "postcode": "ΤΚ",
      "father_name": "Πατρώνυμο",
      "landline_telephone": "2610XXXXXX",
      "mobile_telephone": "69XXXXXXXX"
    }
  ],
  "professors": [
    {
      "name": "Όνομα",
      "surname": "Επώνυμο",
      "email": "email@ceid.upatras.gr",
      "username": "username",
      "password": "password",
      "topic": "Ειδικότητα",
      "department": "Τμήμα",
      "university": "Πανεπιστήμιο",
      "landline": "2610XXXXXX",
      "mobile": "69XXXXXXXX"
    }
  ]
}
Σημείωση: Τα πεδία που είναι υποχρεωτικά είναι: name, surname, email για όλους, student_number για φοιτητές, και topic, department, university για καθηγητές.
